About Project

VectorScanX is a Python-based malware detection tool designed to identify malicious files using YARA rules, SHA-256 hash verification, and quarantine mechanisms. The application helps users detect suspicious files quickly while providing an easy-to-use graphical interface.

Features

⚡ Fast File Scan

Scan files quickly using optimized detection methods.

🛡 Malware Detection

Detect known malware using YARA signatures.

🔐 SHA-256 Hash Check

Verify file integrity using secure hash comparison.

📁 Quarantine

Move suspicious files into quarantine for safety.

💻 User Friendly GUI

Simple desktop interface developed using Python Tkinter.

📊 Detection Report

Generate scan results with malware status.
